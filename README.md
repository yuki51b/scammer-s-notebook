

## ■サービス名
### 詐欺師の手帳
アプリURL: https://scammers-notebook.com/
[![Image from Gyazo](https://i.gyazo.com/7f2c2cfc4ad0b3f2699ed538ab8fc41a.png)](https://gyazo.com/7f2c2cfc4ad0b3f2699ed538ab8fc41a)

## ■サービス概要
詐欺の可能性や対策の提示・被害体験のシェアで詐欺知識を向上し、被害を未然に防ぐ事ができるサービスです

具体的には、
- ユーザーの入力された状況からどんな詐欺の可能性があるのかをAIが診断し、簡単な詳細と対策を提示してくれます。
- 実際に被害にあったユーザーの経験や状況を投稿と閲覧、そしてXへのシェアができます。
- 詐欺知識向上の為に、まず詐欺師視点での詐欺の手口を閲覧できます。さらにLINEでの友達追加をすると詐欺の情報が毎朝7時に一回、通知されます。


## ■このサービスへの思い・作りたい理由

- 自分自身や家族、友人などが詐欺被害に遭った経緯から、詐欺は身近であると同時にYoutubeなどで多くの人が詐欺に引っかかっている現状を知りました。このことから詐欺被害を未然に防ぐサービスを作ってみたいと思いました。
- 「詐欺を立証することが難しいこと」や「詐欺師を捕まえたとしてもお金が返ってくるわけではない」ことなどから、詐欺は詐欺被害の未然防止が大切だと考え、ユーザーの詐欺知識向上に繋がるサービスを意識しました。


## ■ユーザー層について

 ### 実際に詐欺被害にあった方々
  - 詐欺被害に遭った方々は、再度同じ被害を避けたいという強い意識を持ち、詐欺防止のための情報を他の方に共有したいと考えることが多いと考えます。

 ### 詐欺かどうかわからないけど、人に相談するのが苦手な方
  - 詐欺の可能性を疑っていても、弁護士や他人に直接相談することに抵抗を感じる人がいると思います。AIを使った診断ツールは、匿名性が高く、簡単に利用できるため、そうした方々が安心して初期診断を受ける手段として使っていただきたいです。

 ### 現在悪徳マルチなどの勧誘などを受けていて、すぐ調べたいという方
  - 急な勧誘や投資話に対して、即座に情報を確認し、詐欺のリスクを判断することは非常に重要です。AIを利用したリアルタイムの診断ツールは、このようなニーズに対応し、迅速な意思決定を支援できると考えます。

## ■サービスの利用イメージ

- レスポンシブ対応によってPCとスマホの両方から使うことができます。
- ユーザーは「これって詐欺かもしれない」と思ったら、状況をアプリに入力して診断することでどんな詐欺の可能性があるかを知ることができます。
- 詐欺被害について、ユーザーはマークダウン記法を使った投稿と、Xへのシェアができます。

## ■ユーザーの獲得について

### Xへのシェア機能
- 動的OGPによって詐欺名や投稿のタイトルを表示することで、目に止まりやすい様にしました。
### ZennやQiitaへの記事投稿
- 開発記事や、使った機能の記事を投稿することでユーザーの獲得を広げたいと考えています。

## ■サービスの差別化ポイント・推しポイント

- 犯罪被害の中でも「詐欺」というジャンルに絞ったこと。
- 詐欺師視点の**詐欺の手口**を閲覧できること。
- マークダウン記法を取り入れて、ユーザーの投稿の可動性向上に期待できること。
- AIを使った詐欺診断なので、人に相談することが苦手な方も利用しやすいこと。

## ■今後の開発について
### いいね機能の実装
- ユーザーの投稿に対していいねができる実装を考えています
### タグ機能
- 投稿した際の詐欺名がタグとなり、タグでの検索の実装を考えています。
### テストコード
- まだまだテストコードが書けていないので、ガバレッジ率を上げていきたいです。

- ユーザー作成機能
- 詐欺被害を投稿する機能
- 投稿の編集・削除機能
- 詐欺一覧と投稿一覧機能
- 検索機能
- オートコンプリート機能
- 詐欺詳細ページ(詐欺師視点)
- i18nによる国際化
- お問い合わせ(googleフォーム)・利用規約・プライバシーポリシーの作成

本リリース
- LINE認証機能
- LINE通知機能

- マークダウンプレビュー機能
- 検索機能(非同期)
- ページネーション(非同期)
- X(旧Twitter)へのシェア機能(動的OGP)
- 詐欺についての相談先一覧機能
- 独自ドメイン
- マドフくん(アプリキャラ名)の応援メッセージ表示
- 診断中のローディング画面
- 管理者ページログイン機能
- 詐欺情報の新規作成・編集・削除機能(管理者のみ)
